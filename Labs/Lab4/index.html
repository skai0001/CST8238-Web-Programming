    Q2.What is the difference between a relative and absolute measure unit in CSS? Why are relative units preferred over absolute units in CSS ?

  1. Relative units specify a relative unit with respect to another property

  2. Absolute unit are fixed and expressed in the exact size

  3. Relative units are preferred on absolute units ,because relative units scales better between rendering mediums while screen sized varries so much

    Q3.What are the element selector and grouped element selector in CSS?Provide an example of each .

  1. Element selector selects all the instances of a give HTML element. Ex.p { background-color: yellow; }

  2. Grouped element selector combines multiple identical rules into a single rule.Ex. p,div{ margin:0; padding:0;}
